# Gestion Pharmacie

Projet réalisé en JavaFX, Hibernate et MySQL.

## Technologies utilisées

- Java
- JavaFX
- FXML
- Hibernate
- MySQL

## Fonctionnalités

- Gestion des médicaments
- Ajout, modification et suppression des médicaments
- Recherche de médicaments
- Persistance des données avec MySQL
